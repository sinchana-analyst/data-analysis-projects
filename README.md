Store Data Analysis Dashboard (Excel)

Project Overview

This project analyzes store sales data using Microsoft Excel and interactive dashboards.

Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning

Dashboard Features

- Monthly Sales Trend Analysis
- Sales by Gender
- Sales by State
- Customer Age Analysis
- Order Status Breakdown
- Sales Channel Performance

Key Insights

- Amazon generated the highest sales among all channels.
- Most orders were successfully delivered.
- Sales trends varied across different months.
- Customer demographics helped identify major purchasing groups.
